# Solar-system-simulation
A simplified simulation of the solar system
Body_Class.py file contains the body class along with the code needed to run it for the purposes of the experiment, and Body_Class.txt contains the same information but in a txt file format. The bodies.json file contains the data needed to initialise the simulation accurately, particularly the masses and radii of each body. The energy_file is where energy is wrote out to when the energy function is used
